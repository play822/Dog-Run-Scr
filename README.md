 # 👨‍🏫 게임 소개
![DogRun](https://github.com/play822/Dog-Run/assets/61472555/b87ae558-6946-4ddf-9fba-960ce65aa062)


 __치와와를 조종해서 장애물을 피하는 3D 러닝 액션게임__  
 __무한히 이어지는 맵에서 다가오는 장애물들을 피하고__  
 __오래 살아 남아서높은 점수를 달성 하는 것이 이 게임의 목표!__
